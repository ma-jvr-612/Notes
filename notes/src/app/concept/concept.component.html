The concept Add a recipe Step one Recipes are recurring lists that you only need to make once. Think of a recipe like stir-fried rice.
You only need to write the recipe for stir-fried rice once in the recipe section and then you can keep using it in the grocery list tab. Add a grocery list Step two Grocery lists are lists you make everytime you might go to a supermarket. The power is that you can now add recipes with the click of a button.
If you feel like making your stir-fried rice again, just add the recipe to the grocery list without the need to lookup the same recipe online, and you can add other items to the same grocery list as well.
